This is my attempt at trying to learn how LLMs work from scratch and trying to recreate one from scratch as well. I have been extremely interesting in how LLMs work especially after one of the college courses that I had taken. I have been interested in learning to design one on my own for a long time, so this project will attempt to achieve that to some extent.
The sources that I used while learning was the textbook Build a Large Language Model (From Scratch) by Sebastian Raschka and the associated GitHub Repo with it. I will be making notes on things I've learned along the way just to document my notes and have an area for all my notes.
